About Me

After years in the construction industry, I transitioned my passion for design, structure, and problem-solving into front-end development and user experience design. I build intuitive, visually engaging applications using HTML, CSS, JavaScript, and React, with Python and SQL supporting more dynamic, data-driven functionality. That background taught me how to think practically, communicate clearly, and design solutions that actually work in the real world. I’m especially passionate about creating enjoyable, effortless user experiences that also meet business goals. From wireframing and prototyping in Figma to building responsive, accessible interfaces, I focus on thoughtful UX/UI decisions that make products easy to use and pleasant to interact with. I’ve built full-featured React applications, booking-enabled websites, and smaller projects that highlight clean, efficient code and strong usability. I take pride in writing maintainable, well-tested code, using tools like Jest and modern frameworks to ensure reliability and performance. What drives me most is translating good design into interactive solutions that balance user needs with client objectives—delivering products that feel polished, purposeful, and genuinely useful.

OmniList

Welcome to OmniList! I invite you to explore the application by curating your own personal lists of movies, TV shows, and books. This project leverages the browser's Local Storage API to implement client-side persistence, ensuring your data is saved securely on your device across sessions without the need for an external database. As you interact with the application, the 'For You' sections dynamically recalculate based on your stored preferences, demonstrating how local state management can be effectively used to drive a personalized and responsive user experience.

Key Features:

  • Multi-API Integration (TMDB, Google Books, NYT)
  • User Authentication & Data Persistence
  • Custom Watchlists & Reading Lists
  • Responsive Design

CVLMT Website

This full-stack web application was designed and built to streamline the booking and client intake process for a massage therapy business. It features a custom scheduling system integrated with Google Calendar, automated email notifications, and a digital intake form.

Key Features

  • Google Calendar Integration: Real-time availability checking and event creation.
  • Interactive Intake Form: Includes a drawable canvas for clients to mark problem areas.
  • Automated Workflows: Handles PDF generation and SMTP email notifications seamlessly.

Please feel free to make a mock booking! As this is a portfolio project, the business is not yet active, and no payment processing has been enabled. You are invited to test the reservation flow, interactive body charts, and automated workflows without any commitment.
